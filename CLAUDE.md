# Proyecto 06 — Nuevo Miembro AdmiraNext

> Guía de onboarding interactiva para nuevos miembros del equipo AdmiraNext.

## Contexto

Sitio web standalone para facilitar la incorporación de nuevos empleados con una checklist paso a paso. Los nuevos miembros marcan cada punto completado hasta estar listos para trabajar.

**URL pública:** https://csilvasantin.github.io/nuevo-miembro-admiranext/

## Arquitectura

Aplicación web estática basada en checklist interactiva. Estructura simple sin backend requerido.

## Notas para IAs

1. **Propósito**: Herramienta de onboarding ligera y autoservicio — los nuevos miembros completan los pasos por su cuenta.

2. **Interfaz**: Checklist con items marcables. Al terminar todos, el usuario está listo para trabajar.

3. **Publicación**: Vive en GitHub Pages, accesible directamente desde la URL pública.

4. **Integración**: Complementa a AdmiraNext-Team (Proyecto 03) en el flujo de incorporación de personal.

5. **Próximos pasos**: Expandir checklist con más detalles, vincular con inventario de equipos (admira-equipos), agregar feedback o confirmación al completar.
